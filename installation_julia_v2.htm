Guide dinstallation et dutilisation de Julia/Pluto


Ce guide est composé de trois sections. Les deux premières sections sont à faire quune seule fois sur votre ordinateur. La troisième section concerne la procédure générale pour débuter une session Pluto sur votre fureteur (« browser »). Vous pouvez cliquer (ctrl-clic) sur la section dintérêt dans la liste ci-bas pour vous y rendre directement :

 

Installation de Julia

Vous pouvez aussi vous référer au lien suivant pour vous aider : https://plutojl.org/

Cette procédure nest à faire quune seule fois sur votre ordinateur.

1.       Télécharger lapplication Julia pour votre système dopération au lien suivant : https://julialang.org/downloads/.

2.       Installer lapplication en cliquant sur le fichier téléchargé. Par exemple, le ficher exécutable est julia-1.9.2-win64.exe pour  Windows ou julia-1.9.2-mac64.dmg pour macOS en date du 26 juillet 2023. Sélectionner loption « add Julia to PATH » lors de linstallation.

3.       Tester linstallation en ouvrant lapplication Julia. Taper 1 + 1 et peser sur Enter de votre clavier pour entrer la commande. Vous devriez voir la réponse 2 dans votre fenêtre dapplication :

4.      Pour quitter Julia, vous navez quà taper la commande exit() et peser sur Enter. Alternativement, vous pouvez cliquer sur le x de la fenêtre sur Windows (ou léquivalent sur macOS).

Préparation de Julia

Cette procédure doit être suivie quune seule fois sur votre ordinateur.

1.       Ouvrir lapplication Julia.

2.       Dans la console de Julia, taper import Pkg et peser sur Enter

3.       Ensuite, Taper Pkg.add("Pluto") et peser sur Enter. Linstallation peut prendre quelques minutes selon la vitesse de votre internet. Si linstallation est un succès, vous devriez voir le message à la fin tel quindiqué à la figure suivante :

 

 

 

 

Installation des « packages »

Cette procédure doit être suivie quune seule fois sur votre ordinateur.

1.       Ouvrir lapplication Julia (seulement si vous avez fermé la console suite à létape précédente).

2.       Dans la console de Julia, taper import Pluto; Pluto.run()suivi de Enter sur votre clavier.

3.       La page web suivante devrait souvrir dans votre fureteur (« browser »). Dans la section « Open a notebook », copier lURL suivant (lien),  coller dans la zone « Open a notebook » et peser sur « Open ».

4.       Linstallation devrait se faire automatiquement lorsque Pluto ouvrira le fichier. Linstallation est terminée lorsque le bouton « Status » (voir image ci-bas) na plus de bordure jaune. Soyez patient, cette étape devrait prendre plusieurs minutes

 

 

 

Valider linstallation

 

Cette procédure est nécessaire quune seule fois sur votre ordinateur.

1.       Ouvrir lapplication Julia.

2.       Dans la console de Julia, taper import Pluto; Pluto.run()suivi de Enter sur votre clavier. Linterface web suivante devrait souvrir dans votre fureteur (« browser »).

3.       Dans la section « Open a notebook », coller lURL suivant (lien) et peser sur « Open ».

4.       Lire et suivre les indications dans le « notebook » qui souvrira.

 

 

Débuter une session Pluto pour la suite du cours

 

Cette procédure est à faire à chaque réouverture de Julia pour utiliser Pluto dans votre fureteur.

1.       Ouvrir lapplication Julia.

2.       Dans la console de Julia, taper import Pluto; Pluto.run()suivi de Enter sur votre clavier. Linterface web suivante devrait souvrir dans votre fureteur (« browser »). Cette consigne sera inscrite dans les documents vous demandant dutiliser Pluto lorsque nécessaire.

3.       Après votre travail, vous pouvez quitter Pluto à partir de la console Julia avec le raccourcie CTRL-C. Pour fermer la console Julia, vous pouvez cliquer sur le X de la fenêtre ou taper exit() suivi de Enter.
